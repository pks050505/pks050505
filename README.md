## 🚀 Projects & Open Source

### 1. **custom_floating_navigation_bar** (2024–2025) – Open Source Flutter Package
A beautiful, highly customizable **floating navigation bar** widget for Flutter apps.  
Published on [pub.dev](https://pub.dev/packages/custom_floating_navigation_bar)  

**Features**:
- Smooth animations & transitions
- Badge support (text or dot indicators)
- Multiple styles (circular, pill, rectangular)
- Customizable notch/floating button position (center, left, right, or custom)
- Configurable radius, height, gap, outline/filled icons, and more

Perfect for modern apps that want a sleek, floating bottom nav with a FAB-like experience.  
Give it a try → `flutter pub add custom_floating_navigation_bar`  
**pub.dev Link**: [custom_floating_navigation_bar](https://pub.dev/packages/custom_floating_navigation_bar)
